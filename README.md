A class-based flow like front-end architecture.

### package.json

| **Script** | **Description**                  |
| ---------- | -------                          |
| build      | Build KABUDA.js                  |
| dev        | Build KABUDA.js in watch mode    |
| test       | Run tests                        |
| test:watch | Run tests in watch mode          |
| cover      | Run and generate coverage report |
